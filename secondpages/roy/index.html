PROJECT: ROY LICHTENSTEIN—ALL ABOUT ART
Interactive Code | 2017 | College Project

 

The goal of this project is to use SFMOMA's Archives as a source for content and select a past exhibition to create a microsite showcasing the work in that exhibition.

The exhibition I chose was Roy Lichtenstein: All About Art in 2005. In this website, I animated his work along with a parallel effect. I am trying to make the web experience like a comic book.
