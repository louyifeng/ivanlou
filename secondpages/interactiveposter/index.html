PROJECT: INTERACTIVE POSTER—AQUARIUM
Interactive Code | 2017 | College Project

 

“The goal of this project is to publish an impactful ‘poster’ on the web that showcases your strong opinion/perspective on a topic you are passionate about.”

The topic I chose was ecology. I used only CSS&HTML to create this aquarium. As an animal lover, I loving going to zoos, but still I feel sorry for the animals that are imprisoned in the zoo since they lost their freedom. In this interactive poster, I switch the roles between human's and animals' in order to make people pay attention to the importance of freedom to animals.
